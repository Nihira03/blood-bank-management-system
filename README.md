# Blood Bank Management System 🩸

## Overview
A database-driven Blood Bank Management System developed to manage donor records, blood inventory, recipient requests, and blood availability efficiently.

## Features
- Donor Management
- Blood Inventory Tracking
- Recipient Request Handling
- Blood Availability Search
- CRUD Operations
- Database Record Management

## Tech Stack
- MySQL
- DBMS Concepts
- HTML/CSS

## Objectives
- Maintain donor information
- Track blood stock levels
- Process blood requests
- Improve blood bank management efficiency

## Future Enhancements
- Online donor registration
- Real-time inventory updates
- SMS/Email notifications
- Analytics dashboard
